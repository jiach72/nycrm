# Nanyang CRM 功能清单 (Feature List)

本文档汇总了 Nanyang CRM 系统截至目前 (2026-01) 已实现的所有核心功能模块与技术特性。

## 1. 核心仪表盘 (Dashboard 2.0)
采用最新的 **Glassmorphism (毛玻璃)** 设计风格，提供沉浸式的指挥中心体验。

### 1.1 数据可视化
- **关键指标卡片 (KPI Cards)**: 实时展示总线索数、销售转化率、进行中任务等核心数据。支持动态渐变色背景。
- **销售转化漏斗 (Funnel Chart)**: 可视化展示从"访问"到"成交"的各阶段转化率。
- **线索增长趋势 (Trend Chart)**: 折线图展示本周/本月的线索新增趋势。
- **线索来源分布 (Donut Chart)**: 环形图展示线索来源渠道（如搜索引擎、直接访问等）的比例。

### 1.2 效率工具
- **智能日历 (Smart Calendar)**:
    - 集成 `v-calendar`，支持月/周视图。
    - 联动展示 **待办任务 (Tasks)** 和 **预约 (Appointments)**。
    - 任务状态红绿点指示（已完成/未完成）。
- **实时天气 (Live Weather)**: 此模块集成 Open-Meteo API，实时显示新加坡（或指定地点）的温度与天气状况。
- **今日日程 (Timeline)**: 侧边栏时间轴，直观展示当天的会议与行程安排。

### 1.3 实时数据流
- **最新线索**: 表格展示最近录入的潜在客户，支持快速跳转详情。
- **咨询消息**: 实时流式展示来自客户门户 (Customer Portal) 的访客留言。

---

## 2. 线索管理 (Lead Management)
全生命周期的销售线索跟踪与管理。

- **快速录入**: 支持通过弹窗 (Modal) 在任意页面快速创建新线索，无需跳转。
- **状态流转**: 标准化销售流程管理 (New -> Contacted -> Qualified -> Proposal -> Negotiation -> Converted / Lost)。
- **列表视图**: 支持分页、筛选的线索列表。
- **详情视图**: 查看线索完整信息及历史跟进记录。

---

## 3. 任务管理 (Task Management)
基于看板 (Kanban) 的任务协作系统。

- **看板视图**: 
    - 拖拽式 (Drag & Drop) 任务状态变更。
    - 清晰的列式布局 (To Do, In Progress, Review, Done)。
- **列表视图**: 传统的任务清单模式，适合批量处理。
- **任务属性**: 支持设置优先级 (High/Medium/Low)、截止日期、关联线索等。

---

## 4. 报表中心 (Report Center)
深度数据分析与决策支持。

- **综合概览**: 聚合多维度数据的分析面板。
- **交互式图表**: 使用 ECharts 渲染的高性能图表，支持 hover 交互、图例筛选。
- **视觉优化**: 统一的 UI/UX 设计，提供专业的数据展示体验。

---

## 5. 客户门户 (Customer Portal)
面向最终用户的对外展示与互动窗口。

- **品牌展示**: 公司介绍、服务项目展示。
- **在线咨询**: 访客可提交咨询/留言，数据直接打通至 CRM 系统的"咨询消息"模块。
- **响应式设计**: 完美适配桌面与移动端浏览。

---

## 6. 后端与架构 (Backend & Architecture)
系统底层技术栈与基础设施。

- **API 服务**: 基于 Node.js (Express) 构建的 RESTful API。
- **数据持久化**: 使用 Prisma ORM 管理数据库，支持 SQLite/PostgreSQL。
- **鉴权系统**: 基于 JWT (JSON Web Token) 的身份验证与会话管理。
- **业务模型**:
    - `User`: 系统用户与权限。
    - `Lead`: 销售线索。
    - `Task`: 任务与协作。
    - `Appointment`: 会议与预约。
    - `Inquiry`: 客户咨询。

## 7. 系统特性
- **国际化 (i18n)**: 基础架构支持多语言扩展。
- **响应式布局**: 管理后台基于 Element Plus + Flex/Grid 布局，适配不同尺寸屏幕。
- **TypeScript**: 全栈使用 TypeScript 开发，保障代码类型安全与可维护性。
